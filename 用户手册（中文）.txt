# PDF 内容提取工具 - 用户使用手册

## 1. 概述
PDF 内容提取工具是一款安全且完全运行在本地的桌面端应用程序，专为高保真度提取 PDF 文档中的图文资产而设计。它能在无需依赖外部云端处理的前提下，将复杂文档解析为纯文本 Markdown、图文混合 Markdown 以及独立的高清原始图片，确保企业数据的绝对隐私。

## 2. 系统环境要求
- 操作系统：Windows 10 或 Windows 11
- 运行环境：内置启动脚本 `start.bat` 会自动创建隔离的 Python 虚拟环境并配置相关依赖，无需手动安装。

## 3. 安装与初始化
1. 将下载的应用程序压缩包解压至本地的英文/拼音路径目录下。
2. 双击执行 `start.bat`。
3. 系统将自动完成所有环境的配置，并在您的默认网页浏览器中启动图形化操作界面（默认地址通常为 http://localhost:8501）。

## 4. 核心功能操作流
### 4.1 文档上传
请将目标 PDF 文件拖拽至指定的虚线上传区域，或点击此区域在本地资源管理器中选择文件。支持的格式：`.pdf`。

### 4.2 执行解析
点击下方高亮的“开始解析”按钮。程序将根据文档的页面规模与排版复杂度执行深度解析。通常耗时在数秒至几分钟之间，请耐心等待进度条完成。

### 4.3 成果查看与资产导出
解析完毕后，界面中心将呈现三个独立的工作区标签：
- **📄 图文混合区：** 该区域提取了核心文字段落，并在段间精准嵌入了相应的图像。您可以直接跨越文字与图像进行拖拽（或 `Ctrl+A`），复制并粘贴至 Microsoft Word、Notion 等主流富文本编辑器，实现图文一体化转移；亦可点击底部按钮下载包裹完整的 `.md` 文件。
- **📝 纯文本 Markdown 区：** 该区域提供经过去噪处理的纯净版代码文本，剔除了一切无关图片标签与多媒体元素。极为适合作为本地知识库归档或输入至大语言模型（LLM）进行分析。
- **🖼️ 美术素材区：** 一座汇总了页面中所有视觉元素与矢量图形的美术素材库。您可在此单独预览、下载单张素材，或通过一键打包按钮获取包含全尺寸素材的统一 `.zip` 压缩包。

## 5. 异常处理与边缘用例说明 (FAQ)
- **出现黄色警告横幅：** 若界面顶部弹出警告横幅，说明您上传的 PDF 在生成阶段已被设计方实施了“全转曲”（Create Outlines）操作，或者它本质上是一张没有任何底层文本流的扫描图片。受限于文档底层数据结构，系统无法以常规逻辑提取真实文本（需借助含 OCR 技术的引擎处理）。但即便如此，系统的后备截图机制仍将生效，尽最大努力为您提取并分离出页面中的可视化碎片供美术素材区查阅。
- **命令行闪退或无法启动：** 常见原因为工作目录路径中包含非法或罕见字符。建议将软件目录转移至更短的根路径下进行尝试，并检查系统执行策略是否阻拦了批处理文件的运行。
