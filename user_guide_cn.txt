PDF 内容提取工具 - 用户使用手册

1. 概述
PDF 内容提取工具是一款运行在本地的桌面端应用程序，专为精准提取 PDF 文档中的纯文本（Markdown 格式）与高清原始图片（尺寸大于 100px）而设计。所有交互与数据处理均在您的计算机本地完成，不会向外部上传任何文件，以保障数据的绝对隐私与安全。

2. 系统要求
- 操作系统：Windows 10 或 Windows 11
- 运行环境：内置一键运行脚本，只要双击即可自动配置环境（若从源码运行需自行安装 Python 3.8 或更高版本）。

3. 安装与启动
- 解压下载的应用程序压缩包，并打开解压后的主目录。
- 找到并双击运行 “start.bat” 脚本。
- 系统将自动检查并安装必要的依赖项，配置完成后会在您的默认浏览器中直接打开本应用程序的交互界面。

4. 核心功能及操作步骤
步骤 4.1：上传文档
- 当应用程序在浏览器中就绪后，您可以直接将需要提取的 PDF 文件拖拽至指定的上传区域，或点击上传区域从您的本地目录中选择文件。
- 文件上传成功后，系统会即时提示并显示该文件的大小。

步骤 4.2：执行解析
- 点击页面下方显眼的“开始解析”按钮。
- 程序将开始工作，请耐心等待。处理时间取决于文件的大小与复杂程度（通常需要数十秒到分钟不等）。

步骤 4.3：查看与下载结果
- Markdown 文本提取：解析完成后，点击“📝 Markdown 文本区”选项卡以查看提取出的文本内容。您可以通过点击下载按钮一键将其保存为本地的 `.md` 文件，或直接在页面中展开代码块查看/复制原始 Markdown 代码。
- 高清图片提取：点击“🖼️ 美术素材区”选项卡以查看从 PDF 中提取到的所有图片。您可以单独下载列表中的某张图片，也可以通过点击“📦 一键打包下载所有图片 (.zip)”按钮将全部图片以 ZIP 压缩包的形式一次性下载。请注意，尺寸小于 100 像素的微小图像以及背景纹理已被系统内置的过滤机制自动剔除。

5. 常见问题解答与排查 (FAQ)
- 问题：控制台报错或程序闪退无法启动。
  解决：请检查操作系统是否阻止了脚本的运行，并确保解压路径中不要包含罕见的特殊字符。
- 问题：运行 “start.bat” 后浏览器没有自动打开。
  解决：这是偶发的系统配置问题。当终端提示程序已经在运行（例如 “You can now view your Streamlit app in your browser”），您可以手动在浏览器地址栏中输入访问 http://localhost:8501。
- 问题：解析出的 Markdown 格式有些错乱。
  解决：PDF 是一种基于绝对定位的文档格式，其本身并不包含严谨的语义结构。工具会尽最大可能还原其段落排版，但对于复杂的表格和多列排版，提取后可能需要少量的手动修正。
