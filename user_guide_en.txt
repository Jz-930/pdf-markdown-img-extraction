# PDF Content Extractor - User Guide

## 1. Overview
The PDF Content Extractor is a secure, locally-hosted desktop application designed for high-fidelity extraction of text and images from PDF documents. It parses documents into pure Markdown, inline mixed Markdown, and discrete high-resolution image assets without relying on external cloud processing.

## 2. System Requirements
- Operating System: Windows 10 or Windows 11
- Environment: The provided `start.bat` script automatically creates an isolated Python virtual environment and installs all required dependencies.

## 3. Installation & Initialization
1. Extract the application archive to a preferred local directory.
2. Execute `start.bat`. 
3. The system will automatically configure the required environment and launch the graphical interface in your default web browser (typically at http://localhost:8501).

## 4. Operation Workflow
### 4.1 Uploading Documents
Drag and drop your target PDF file into the upload zone, or click to browse your local file system. Supported format: `.pdf`.

## 5. Troubleshooting & Edge Cases
- **Warning Banner Displayed:** If a yellow warning banner appears, it indicates the PDF was likely converted to outlines (Create Outlines) prior to export, or is a flattened scanned image. These documents lack underlying text streams and require external Optical Character Recognition (OCR) tools for text extraction. However, the system's fallback mechanisms will still attempt to isolate and extract all visual elements into the Artwork Assets tab.
- **Application Fails to Launch:** Ensure the directory path does not contain special characters or spaces. Verify your system's execution policies permit running batch scripts.
