PDF Content Extractor - User Guide

1. Overview
The PDF Content Extractor is a local desktop application designed to accurately extract raw text (in Markdown format) and high-resolution images (>100px) from PDF documents. All processing is performed locally to ensure data privacy and security.

2. System Requirements
- Operating System: Windows 10/11
- Environment: Python 3.8 or higher installed (if running from source) or via the provided launcher.

3. Installation and Startup
- Locate the unzipped application directory.
- Double-click the "start.bat" script. This will automatically set up the environment, install required dependencies, and launch the application in your default web browser.

4. How to Use
Step 4.1: Uploading a Document
- Once the application opens in your browser, drag and drop your target PDF file into the designated upload area, or click the upload area to select a file from your computer.
- A success message will appear once the file is uploaded.

5. Troubleshooting
- If the application fails to start, ensure that Python is properly installed and added to your system's PATH.
